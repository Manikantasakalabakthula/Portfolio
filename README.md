## About
Simple portfolio site showcasing projects, contact info and a short about section.  
Focus: lightweight, responsive layout and a CSS-only theme toggle (light / dark) that requires no JavaScript.

## Features
- Responsive layout (desktop / tablet / mobile)
- CSS-only theme toggle (uses a hidden checkbox and CSS variables)
- Hamburger menu for small screens
- Projects section with live demo and GitHub links
- Contact links: email, LinkedIn, phone
- No JavaScript required for core UI/UX
